# Resolución de conflictos

Cuando dos personas editan la misma línea de un archivo, ocurre un conflicto.

## Pasos para resolver:
1. git pull (verás el conflicto)
2. Edita el archivo para dejar solo la versión correcta
3. git add archivo
4. git commit -m "Conflicto resuelto"
