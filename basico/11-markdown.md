# Markdown en GitHub

GitHub usa Markdown para dar formato a archivos como README.md.

## Ejemplos:
- **Negritas**: **texto**
- _Cursivas_: _texto_
- Listas: - item
- Enlaces: [GitHub](https://github.com)
- Código: `console.log('Hola')`

# Consejos
- Usa títulos y subtítulos (#, ##, ###)
- Agrega imágenes y enlaces
