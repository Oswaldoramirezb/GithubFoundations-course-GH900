# Clonar un repositorio

Para obtener una copia de un repositorio remoto:

- git clone URL

Esto descarga todos los archivos y el historial de cambios.

# Fork y Pull Request

- Fork: Crea una copia de un repositorio en tu cuenta de GitHub.
- Pull Request: Solicita que tus cambios sean integrados en el repositorio original.

# Buenas prácticas

- Escribe mensajes de commit claros.
- Usa ramas para nuevas funcionalidades.
- Haz pull antes de push para evitar conflictos.
- Mantén tu repositorio actualizado.
