# Licencias en GitHub

Es importante definir una licencia para tu proyecto.

## Ejemplos comunes:
- MIT
- GPLv3
- Apache 2.0

Puedes agregar un archivo LICENSE al repositorio. GitHub ofrece plantillas al crear un nuevo repositorio.

# ¿Por qué usar licencia?
- Define permisos de uso
- Protege tu trabajo
- Facilita la colaboración
