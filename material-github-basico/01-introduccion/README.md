# Introducción a GitHub

GitHub es una plataforma para alojar código, colaborar en proyectos y gestionar versiones usando Git.

## ¿Qué es Git?
Git es un sistema de control de versiones que permite llevar un historial de los cambios en los archivos de un proyecto.

## ¿Qué es GitHub?
GitHub es un servicio en la nube que permite almacenar repositorios Git y colaborar con otros desarrolladores.

## ¿Por qué usar GitHub?
- Control de versiones
- Colaboración
- Visibilidad de proyectos
- Automatización (CI/CD)
