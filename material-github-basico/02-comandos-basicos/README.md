# Comandos básicos de Git y GitHub

## Configuración inicial
- git config --global user.name "Tu Nombre"
- git config --global user.email "tu@email.com"

## Crear repositorio
- git init

## Guardar cambios
- git add archivo.txt
- git commit -m "Mensaje"

## Ver historial
- git log

## Subir a GitHub
- git remote add origin URL
- git push -u origin main
